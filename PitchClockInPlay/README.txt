Pitch Clock in Play - Academic Project
Analysis of Pitcher Performance Following the 2023 Pitch Clock implmentation

Course: SDS 336 - Practical Machine Learning
Tools: Python, Pandas, Scikit-lean, Seaborn
Semester: Fall 2024

Description: Project aims to observe and provide context for what areas of pitcher performance were most impacted by the pitch clock's intoduction in the 2023 season. By using 2022 and 2023 Player and Pitch Data, we look at performance on a pitch by pitch, game and season basis to analyze differences in the two years. 

Note: This project was completed in Fall 2024 as part of a group project. Original datasets used in analysis are no longer accessible and project code is altered to remove unncessary steps not relevant to analysis
